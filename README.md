# Leyla´s Code – Ads Verification Site

GitHub Pages site that hosts `ads.txt` and `app-ads.txt` for AdMob/Adsterra
verification of the **Leyla´s Code** Android app and the
[miner-app.com](https://miner-app.com) website.

## Why this exists
Google AdMob requires every publisher to declare authorised sellers via
[`app-ads.txt`](https://support.google.com/admob/answer/9363762) on the
developer site that is linked from the Play Store listing.
This repository simply serves that file.

## Files
| File           | Purpose                                      |
| -------------- | -------------------------------------------- |
| `ads.txt`      | Authorised digital sellers for the website   |
| `app-ads.txt`  | Authorised sellers for the Android app       |
| `index.html`   | Minimal landing page (required by GH Pages)  |
| `CNAME`        | Custom domain mapping (optional)             |
